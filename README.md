Kuro Bot 🤖
Kuro Bot is a personal assistant bot designed to work on Discord and Telegram. It can perform various tasks like fetching news, providing weather updates, and understanding humanized commands using AI. This project is written in Python and uses free tools and services.

Features 🌟
Humanized Commands: Kuro can understand natural language commands in Portuguese, such as:

"Kuro, como está o tempo?" (Kuro, how is the weather?)

"Kuro, me conte as notícias." (Kuro, tell me the news.)

News Updates: Fetch the latest news using the NewsAPI.

Weather Updates: Get real-time weather information for any city using OpenWeatherMap.

AI-Powered: Uses OpenAI GPT or Hugging Face models to understand and respond to commands.

Multi-Platform: Works on both Discord and Telegram.

Prerequisites 📋
Before running the bot, make sure you have the following:

Python 3.8 or higher: Download and install Python from python.org.

Bot Tokens:

Discord Bot Token: Create a bot on the Discord Developer Portal.

Telegram Bot Token: Create a bot using BotFather on Telegram.

API Keys:

NewsAPI Key: Sign up at NewsAPI.

OpenWeatherMap Key: Sign up at OpenWeatherMap.

OpenAI API Key (optional): Sign up at OpenAI.
